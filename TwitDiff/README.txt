Twitdiff
AntiTree - 2013

SETUP:
- get your consumer keys from dev.twitter.com - create a new application if you don't already have one
- copy the consumer values into the top of the twitget file
- run the twitget file for the first time
- it will prompt you to authorize the application 
- copy the code from the site into the running script
- copy the OAUTH secret values into the script
- create a fake gmail address
- copy the username and password into the top of TwitDiff.py

USAGE: 
- run the twitget as a cronjob for a regular interval (ie ever hour)
- supply it a username to keep track of
- Set another cronjob (daily, weekly) for twitdiff
